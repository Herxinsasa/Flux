# Release Notes

## 1.0.1 (2026-05-17)

### Added
- 内置 Skill（flux-core、problem-summary、analysis-report）打包入 asar
- 设置页”自定义”提供商新增”协议类型”下拉，可在 OpenAI Chat Completions 与 Anthropic Messages 之间切换。
- 协议下拉选项新增端点提示（`/v1/chat/completions`、`/v1/messages`），便于快速校验对接协议。

### Changed
- 导出分析报告成功后，自动刷新当前工作区文件列表，导出文件可立即在文件树中可见。
- 自定义协议类型会参与“测试连接”和“保存配置”，不再固定按单一协议发送请求。
- 设置页下拉触发器视觉优化：箭头样式与大小重新调整，提升识别度并减少突兀感。

### Fixed
- 修复导出报告后“已保存但工作区未刷新”的问题。
- 修复部分自定义配置在回显后协议设置项不易出现的问题（显示逻辑已放宽）。

## 1.0.0 (Released)

### Added
- 帮助菜单新增“关于”入口。
- 新增 About 弹窗：显示应用图标、当前版本与 website 链接。
- 新增 `app:get-version` IPC 通道，渲染进程可读取主进程版本号。
- 提供三栏工作台：文件管理、编辑器、AI 对话。
- 支持 Skill 导入与调用、Provider 配置、Markdown 报告导出。

### Changed
- AI 对话输入支持最多 5 条历史输入（方向键上下切换）。
- 历史切换时抑制 `/skill` 与 `@` 弹窗，避免卡住；重新输入后恢复弹出。
- 应用改为单实例模式：第二次启动弹窗提示“已在运行”，并自动退出。
- 新建文件默认名统一为 `untitled.md`。
- 新建时覆盖已有文件会先清空目标内容，避免残留旧文本。

### Fixed
- 左侧工作集支持关闭已打开文件（`×`）。
- 首个发布版本。
